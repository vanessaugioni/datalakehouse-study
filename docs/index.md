# DataLakehouse Study

Este projeto apresenta um ambiente local para estudar Apache Spark com Delta Lake e Apache Iceberg.

## Objetivo

- Criar um ambiente PySpark local com suporte a Delta Lake e Apache Iceberg.
- Documentar o processo de instalação e configuração.
- Fornecer notebooks de demonstração para aprendizado prático.

## Navegação

- **Apache Spark**: conceitos básicos, configuração PySpark e uso local.
- **Delta Lake**: fluxo Delta Lake, escrita, leitura e time travel.
- **Apache Iceberg**: criação de tabelas, inserção de dados e evolução de schema.
